# AML Transaction Monitoring Project

This project demonstrates Anti-Money Laundering (AML) analysis using Excel and MySQL.

## Tools Used
- Microsoft Excel
- MySQL Workbench
- GitHub

## Excel Work
- Transaction monitoring dashboard
- Suspicious transaction alerts
- Country-wise risk analysis
- Large transaction tracking

## SQL Work
- Created AML database
- Inserted transaction records
- Queried suspicious transactions
- High-value transaction detection
- Country transaction summaries
- Top transaction analysis

## Skills Demonstrated
AML Monitoring, SQL, Excel, Data Analysis, Risk Detection, Reporting
